## 0.11.0 - 2026-06-17

Feature release introducing a platform whitelist for focused workspace declutter.

### Features

- Add a **Platform Whitelist** card in Settings to select which platforms to display; unchecked platforms are hidden everywhere — sidebar, skill cards, install dialogs, install status, and search results.
- Whitelist applies as a full visibility filter across all platform surfaces (16 consumption points), persisted locally with no backend changes.

### Improvements

- Whitelist filtering is reactive: toggling a platform in Settings updates the sidebar and cards instantly.
- The platform whitelist selector lists every install-target platform (built-in + custom) for complete control.
- Align the version shown on the About page with the actual app version.

### Fixes

- Resolve stale About-page version display.

### Tests

- Add coverage for the whitelist filter helper, settings persistence (round-trip + corrupt-JSON tolerance), store derivation, and the Settings whitelist card interactions.

## 0.11.0 - 2026-06-17（中文）

聚焦「平台白名单」的功能版本，用于精简工作区、只保留关注的平台。

### 新功能

- 设置页新增**「平台白名单」**卡片，勾选需要展示的平台；未勾选的平台在所有界面隐藏——侧边栏、技能卡片、安装对话框、安装状态、搜索结果。
- 白名单作为完整的可见性过滤器作用于全部平台展示点（16 处），仅本地持久化，后端零改动。

### 改进

- 白名单过滤为响应式：在设置页勾选/取消即时更新侧边栏与卡片。
- 白名单选择器列出全部安装目标平台（内置 + 自定义），可完整控制。
- 对齐「关于」页显示的版本号与实际应用版本。

### 修复

- 修复「关于」页版本号长期滞后的问题。

### 测试

- 新增白名单过滤工具、设置持久化（往返 + 损坏 JSON 容错）、store 派生、设置页白名单卡片交互的测试覆盖。
